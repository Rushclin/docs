# Démarrage rapide

Ce guide vous permettra d'effectuer votre premier appel API Genuka en moins de 10 minutes.

## Étape 1 : Prérequis

Avant de commencer, assurez-vous d'avoir :
- Un environnement de développement configuré
- Un outil pour tester les API (Postman, curl, ou votre IDE)
- Une connexion internet stable

## Étape 2 : Obtenir vos identifiants

<Note>
Pour le moment, la plateforme développeur n'est pas encore disponible. Contactez-nous à [contact@genuka.com](mailto:contact@genuka.com) pour obtenir vos clés API de test.
</Note>

Une fois vos identifiants obtenus, vous aurez :
- `CLIENT_ID` : Votre identifiant client
- `CLIENT_SECRET` : Votre clé secrète (à garder confidentielle)
- `API_BASE_URL` : L'URL de base de l'API

## Étape 3 : Authentification

### Obtenir un token d'accès

```bash
curl -X POST https://api.genuka.com/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "client_id": "VOTRE_CLIENT_ID",
    "client_secret": "VOTRE_CLIENT_SECRET",
    "grant_type": "client_credentials"
  }'
```

**Réponse attendue :**
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600
}
```

## Étape 4 : Votre premier appel API

Utilisez le token obtenu pour faire un appel API :

```bash
curl -X GET https://api.genuka.com/v1/me \
  -H "Authorization: Bearer VOTRE_ACCESS_TOKEN" \
  -H "Content-Type: application/json"
```

**Réponse attendue :**
```json
{
  "status": "success",
  "data": {
    "client_id": "your-client-id",
    "permissions": ["read", "write"],
    "rate_limit": {
      "limit": 1000,
      "remaining": 999
    }
  }
}
```

```javascript Node.js
const axios = require('axios');

class GenukaClient {
  constructor(clientId, clientSecret, baseUrl = 'https://api.genuka.com') {
    this.clientId = clientId;
    this.clientSecret = clientSecret;
    this.baseUrl = baseUrl;
    this.accessToken = null;
  }

  async authenticate() {
    try {
      const response = await axios.post(`${this.baseUrl}/auth/token`, {
        client_id: this.clientId,
        client_secret: this.clientSecret,
        grant_type: 'client_credentials'
      });
      
      this.accessToken = response.data.access_token;
      return this.accessToken;
    } catch (error) {
      throw new Error(`Authentication failed: ${error.message}`);
    }
  }

  async getProfile() {
    if (!this.accessToken) {
      await this.authenticate();
    }

    try {
      const response = await axios.get(`${this.baseUrl}/v1/me`, {
        headers: {
          'Authorization': `Bearer ${this.accessToken}`,
          'Content-Type': 'application/json'
        }
      });
      
      return response.data;
    } catch (error) {
      throw new Error(`API call failed: ${error.message}`);
    }
  }
}

// Usage
const client = new GenukaClient('your-client-id', 'your-client-secret');
client.getProfile()
  .then(profile => console.log('Profile:', profile))
  .catch(error => console.error('Error:', error));
```

```python Python
import requests
import json

class GenukaClient:
    def __init__(self, client_id, client_secret, base_url="https://api.genuka.com"):
        self.client_id = client_id
        self.client_secret = client_secret
        self.base_url = base_url
        self.access_token = None
    
    def authenticate(self):
        url = f"{self.base_url}/auth/token"
        data = {
            "client_id": self.client_id,
            "client_secret": self.client_secret,
            "grant_type": "client_credentials"
        }
        
        response = requests.post(url, json=data)
        response.raise_for_status()
        
        self.access_token = response.json()["access_token"]
        return self.access_token
    
    def get_profile(self):
        if not self.access_token:
            self.authenticate()
        
        url = f"{self.base_url}/v1/me"
        headers = {
            "Authorization": f"Bearer {self.access_token}",
            "Content-Type": "application/json"
        }
        
        response = requests.get(url, headers=headers)
        response.raise_for_status()
        
        return response.json()

# Usage
client = GenukaClient("your-client-id", "your-client-secret")
try:
    profile = client.get_profile()
    print("Profile:", json.dumps(profile, indent=2))
except Exception as error:
    print(f"Error: {error}")
```

```php PHP
<?php

class GenukaClient {
    private $clientId;
    private $clientSecret;
    private $baseUrl;
    private $accessToken;

    public function __construct($clientId, $clientSecret, $baseUrl = 'https://api.genuka.com') {
        $this->clientId = $clientId;
        $this->clientSecret = $clientSecret;
        $this->baseUrl = $baseUrl;
    }

    public function authenticate() {
        $url = $this->baseUrl . '/auth/token';
        $data = [
            'client_id' => $this->clientId,
            'client_secret' => $this->clientSecret,
            'grant_type' => 'client_credentials'
        ];

        $options = [
            'http' => [
                'header' => "Content-type: application/json\r\n",
                'method' => 'POST',
                'content' => json_encode($data)
            ]
        ];

        $context = stream_context_create($options);
        $result = file_get_contents($url, false, $context);
        
        if ($result === FALSE) {
            throw new Exception('Authentication failed');
        }

        $response = json_decode($result, true);
        $this->accessToken = $response['access_token'];
        
        return $this->accessToken;
    }

    public function getProfile() {
        if (!$this->accessToken) {
            $this->authenticate();
        }

        $url = $this->baseUrl . '/v1/me';
        $options = [
            'http' => [
                'header' => [
                    "Authorization: Bearer " . $this->accessToken,
                    "Content-Type: application/json"
                ],
                'method' => 'GET'
            ]
        ];

        $context = stream_context_create($options);
        $result = file_get_contents($url, false, $context);
        
        if ($result === FALSE) {
            throw new Exception('API call failed');
        }

        return json_decode($result, true);
    }
}

// Usage
$client = new GenukaClient('your-client-id', 'your-client-secret');
try {
    $profile = $client->getProfile();
    echo "Profile: " . json_encode($profile, JSON_PRETTY_PRINT);
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
?>
```

## Vous rencontrez des problèmes ?

<AccordionGroup>
  <Accordion title="Erreur 401 - Unauthorized">
    Vérifiez que vos `CLIENT_ID` et `CLIENT_SECRET` sont corrects et que votre token n'a pas expiré.
  </Accordion>
  
  <Accordion title="Erreur 403 - Forbidden">
    Votre compte n'a peut-être pas les permissions nécessaires pour accéder à cette ressource.
  </Accordion>
  
  <Accordion title="Erreur 429 - Too Many Requests">
    Vous avez dépassé la limite de taux. Attendez avant de refaire une requête.
  </Accordion>
</AccordionGroup>

Pour plus de détails, consultez notre [guide de gestion des erreurs](/getting-started/handling-errors).
