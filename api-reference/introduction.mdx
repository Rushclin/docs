## Prerequis 

Avant toute manipulations, vous devez disposer d'un compte developpeur sur la plateforme [Genuka](www.genuka.com) 

<Note>
La plateforme n'est pas encore disponible, se referer a la CLI pour le faire.
</Note>

## Comment ca fonctionne ?

## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
